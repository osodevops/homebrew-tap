# OSO Homebrew Tap

This is the official Homebrew tap for OSO tools.

## Installation

```bash
brew tap osodevops/tap
brew install kafka-backup
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| kafka-backup | High-performance Kafka backup and restore tool with PITR support |

## Updating

```bash
brew update
brew upgrade kafka-backup
```
